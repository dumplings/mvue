# MVUE

> A Vue.js Component


## 文档
[Page 分页](https://github.com/jiakeqi/mvue/tree/master/src/components/page/README.md)

## 安装
``` shell
npm install mvue -S
```
## 使用
``` JavaScript
import {Page} from 'mvue'
Vue.use(Page)
```
## 编译

``` bash
# 安装依赖
npm install

# 运行项目 localhost:8080
npm run dev

# 打包 dist 文件
npm run prod

```
